# Network-Intrusion-Detection-System
This Network Intrusion Detection System uses Mutual Information to select 20 key features from the NSL-KDD dataset, removing redundancies above 0.95 correlation. Its LSTM model (128 units, dual Dropout) achieved 98.8% accuracy and 0.99 precision/recall, outperforming Random Forest (97.6%) and Decision Trees (97.3%).
